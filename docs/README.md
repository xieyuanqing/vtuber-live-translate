## 当前产品边界

为避免旧文档造成误解，当前 v2.0.7 以以下规则为准：

- 产品是通用实时翻译工具，VTuber 只是视频模式的一种场景。
- 主导航固定为“同传 / 视频 / 历史 / 设置”。
- 同传与视频的长期方案、草稿和本场上下文严格隔离。
- 方案只保存语言、场景和方案提示词，不包含本场临时上下文。
- 本场上下文只在对应业务主页编辑，并注入本次会话快照。
- 术语库已从当前产品入口和提示词链路移除。
- 历史仅自动保存到 App 私有目录，不写公共 Downloads。
- 当前只提供个人 Debug 构建，不建设账号、后端或商店分发。

## 文档维护约定

- 当前行为变化时，同步更新根目录 `README.md`、本索引和相关技术文档。
- 每次功能或重要修复都在 `04-dev-log.md` 顶部追加真实验证记录。
- 历史日志保留当时原文，不用当前设计反向改写旧版本记录。
- 过时计划应明确标记为历史资料，而不是直接删除决策背景。
- 文档、注释和提交说明使用中文；类名、协议字段和命令保留原文。
