## 当前产品边界

为避免旧文档造成误解，当前产品以根目录 `README.md` 和 `CLAUDE.md` 为准：

- 产品是通用实时翻译工具，VTuber 只是视频模式场景库中的一个初始化模板。
- 主导航固定为“同传 / 视频 / 历史 / 设置”。
- 同传与视频的场景库、默认场景、运行时草稿和本场上下文严格隔离。
- 场景库是场景提示词的运行时真源；内置模板只负责首次初始化和用户主动恢复，所有条目都可编辑。
- 场景库是唯一可复用配置层；旧方案库已并入场景库，不再保留命名方案、方案级额外提示词或独立方案编辑器。
- 语言方向不进入场景库，只保存在每个模式各自的运行时草稿中。
- 本场上下文只在对应业务主页编辑；启动时冻结完整 Prompt 和场景名称。
- 术语库已从当前产品入口和提示词链路移除。
- 历史仅自动保存到 App 私有目录，不写公共 Downloads。
- 默认路径是用户填写个人 API Key 的纯本地 Debug 构建，不依赖后端。
- 好友邀请网关是可选的自托管分享扩展，不是账号体系；当前不建设注册、登录、订阅、云同步或商店分发能力。

## 文档维护约定

- 当前行为变化时，同步更新根目录 `README.md`、本索引和相关技术文档。
- 每次功能或重要修复都在 `04-dev-log.md` 顶部追加真实验证记录。
- 历史日志保留当时原文，不用当前设计反向改写旧版本记录。
- 过时计划应明确标记为历史资料，而不是直接删除决策背景。
- 文档、注释和提交说明使用中文；类名、协议字段和命令保留原文。
